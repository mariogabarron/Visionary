# Datos a guardar del usuario a nivel de base de datos.
## Proporcionados directamente por el usuario:
- Nombre (name)
- Email (email)

## Proporcionados indirectamente por el usuario:
- Fecha de registro (registered_on, registered_on_tz, registered_on_tz_offset)
- Fecha del último inicio de sesión. (last_login, last_login_tz, last_login_tz_offset)
- Si la cuenta está eliminada (deleted)
- País (country). Este se podrá editar próximamente
